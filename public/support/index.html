Technical Support

How do I install my workflow in n8n? ‚ñº

To install your workflow in n8n:

  1. Open your n8n instance
  2. Click "Import from JSON" in the workflow editor
  3. Upload the workflow file we provided
  4. Configure any required credentials (API keys, OAuth)
  5. Test the workflow with sample data
  6. Activate the workflow when ready

Watch our included video tutorial for step-by-step guidance.

What if my workflow doesn't work? ‚ñº

All workflows are tested before delivery, but if you encounter issues:

  1. Check that all required credentials are configured
  2. Verify input data format matches the workflow requirements
  3. Review the testing report for any special notes
  4. Contact our support team with error details

We provide free troubleshooting and fixes for 30 days after delivery.
